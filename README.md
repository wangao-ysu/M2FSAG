This is an example file on how the M2FSAG program [1] could be used.

Experiments can be started by running the `demo` script.

Please feel free to contact Ao Wang (awang@stumail.ysu.edu.cn), if you have any problem about this program.

M2FSAG borrows partially from ML-TSK FS [2] and ML-HDTSK FS [3], if you find useful please consider citing below refernces.

[1] G. Du, A. Wang, H. Wu, P. Lu, J. Long, H. Wu， and J. Zhang. Missing multi-label learning with TSK fuzzy system and adaptive graph. IEEE Transactions on Fuzzy Systems, 2026, Accepted.  

[2] Q. Lou, Z. Deng, Z. Xiao, K.-S. Choi, and S. Wang. Multilabel TakagiSugeno-Kang fuzzy system. IEEE Transactions on Fuzzy Systems,2022,30(9):3410–3425.

[3] Z. Bian, Q. Chang, J. Wang, W. Pedrycz, and N. R. Pal. Takagi-sugenokang fuzzy systems for high-dimensional multilabel classification. IEEE Transactions on Fuzzy Systems, 2024,32(6):3790–3804.
